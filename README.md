# Sliding Navigation

A responsive navigation bar with a smooth sliding highlight, built with HTML, CSS, and a touch of vanilla JavaScript. No frameworks.

## Features

- Pure CSS sliding indicator (radio input + `:checked` trick)
- Pill or underline indicator style, toggleable
- Top nav or sidebar layout, toggleable
- Notification badge, hash-based routing, swipe support on mobile
- Remembers your last tab and settings via `localStorage`
- Fully responsive and keyboard accessible

## Usage

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Files

```
index.html   Markup
styles.css   Layout, theme, animations
script.js    Routing, swipe, persistence
```
